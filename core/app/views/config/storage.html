Storage

days
Defines how long the trace should be retained in the H2 data file. Delete All will clear all traces from the H2 data file.
Trace detail data (spans and profiling data) is stored to disk as LZF-compressed json data in a rolling data file to efficiently cap the growth of this large data. (Trace summary data - headline, capture time, duration, user, attributes, metrics - is stored to an H2 data file by the embedded H2 database engine.)

{{dataDir}}

The data directory where the H2 data file and the rolling database are stored. The default value is the directory where the glowroot.jar resides. This can only be changed on the command line, using -Dglowroot.data.dir=<folder>.
