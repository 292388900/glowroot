Storage

Defines how long aggregated transactions should be retained. Aggregated query and profile data may expire sooner than this, see Capped database size below.
Defines how long traces should be retained. Trace profile and entries detail may expire sooner than this, see Capped database size below.
Defines how long gauge data should be retained.
Aggregated query data, aggregated profile data, trace entries and trace profiles can be large, and are stored in a capped database for efficient access and capped disk usage. Once this capped database grows to its maximum size, new data start overwriting the oldest data.
